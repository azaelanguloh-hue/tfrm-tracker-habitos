# Instrucciones (GitHub → Streamlit Cloud)

## 1) Crear repositorio
GitHub → + → New repository  
Nombre sugerido: `tfrm-tracker-habitos`

## 2) Subir archivos
En el repo → botón **+** (junto a Code) → Upload files  
Sube:
- app.py
- habitos_config.json
- requirements.txt
- README.md

Commit changes.

## 3) Publicar (link)
Streamlit Cloud → New app  
Repo: `tfrm-tracker-habitos`  
Main file: app.py  
Deploy → comparte link

## 4) Cambiar frases / hábitos
Edita `habitos_config.json`.

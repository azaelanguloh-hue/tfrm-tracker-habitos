# TFRM — Tracker de Hábitos (Streamlit)

Mini-app simple para usuarias 40–60:
- Mes 1 / Mes 2
- Objetivo del mes
- Checklist de hábitos por día (1–31)
- Botón grande: Guardar mi día (con frase motivadora)
- Vista mensual simple

## Ejecutar
```bash
pip install -r requirements.txt
streamlit run app.py --server.enableCORS false --server.enableXsrfProtection false
```
